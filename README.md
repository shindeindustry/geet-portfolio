## 🚀 Projects

### 1. Jarvis AI Assistant
A Python-based intelligent voice assistant inspired by Iron Man's Jarvis.

Features:
- Voice command processing
- PC automation
- Application control
- Speech synthesis
- AI response generation

Technologies used:
Python, SpeechRecognition, TTS, APIs

---

### 2. Gesture Control System
A computer vision project that allows users to control a computer using **hand gestures**.

Features:
- Real-time hand tracking
- Cursor movement using hand landmarks
- Click detection using finger distance
- Smooth cursor control using filtering

Technologies used:
OpenCV, MediaPipe, Python

---

### 3. Quantum Computing Experiments
Simulation of quantum circuits using the Qiskit framework.

Experiments include:
- Quantum superposition
- Measurement probability experiments
- Quantum circuit simulations

Technologies used:
Python, Qiskit
